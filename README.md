# Linux VM + PostgreSQL + SSH Setup

Учебный мини-проект по администрированию Linux-сервера и настройке PostgreSQL в виртуальной среде.

## Что сделано

- Развернут **Ubuntu Server 24.04** в VirtualBox
- Настроено подключение общей папки между хостом (Windows) и VM
- Настроена сеть (NAT) + исправлены DNS проблемы
- Установлен и запущен **PostgreSQL**
- Импортирована учебная база данных *«Авиаперевозки»*
- Установлен и включен **SSH**
- Настроен проброс порта → подключение по SSH с хоста
